* Пополнение счета (deposit):
  http://0.0.0.0:3000/cash_machine?method=deposit&amount=AMOUNT

  Пример для пополнения счета на 50:
  http://0.0.0.0:3000/cash_machine?method=deposit&amount=50


* Вывод денег (withdraw):
  http://0.0.0.0:3000/cash_machine?method=withdraw&amount=AMOUNT

  Пример для снятия 30:
  http://0.0.0.0:3000/cash_machine?method=withdraw&amount=30


* Проверка текущего баланса (balance):
  http://0.0.0.0:3000/cash_machine?method=balance
